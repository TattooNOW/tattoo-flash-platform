# Affiliate Publishing Folder — Google Drive Structure

This folder structure is auto-created by the `post-show-processing` n8n workflow for each episode.

## Root Folder

```
TattooNOW Show — Affiliate Assets/
```

## Per-Episode Folder

Created for each published episode:

```
Episode [###] — [Title]/
├── Social Graphics/
│   ├── episode-announcement-1080x1080.png      (Feed post)
│   ├── episode-announcement-1080x1920.png      (Story/Reel cover)
│   ├── guest-spotlight-1080x1080.png           (Guest feature post)
│   └── quote-card-1080x1080.png                (Best quote overlay)
├── Video Clips/
│   ├── hook-clip-9x16.mp4                      (15-30s teaser)
│   ├── guest-spotlight-9x16.mp4                (60-90s guest intro)
│   └── tech-tip-9x16.mp4                       (60-90s feature demo)
├── Written Content/
│   ├── episode-summary.md                      (Blog-ready summary)
│   ├── social-captions.md                      (Platform-specific captions + hashtags)
│   └── newsletter-blurb.md                     (Email-ready paragraph)
├── Partner Links/
│   ├── affiliate-links.md                      (Each partner's unique URLs)
│   └── utm-tracking-guide.md                   (How to use tracking params)
└── README.md                                    (What's in this folder + usage guidelines)
```

## Asset Generation Tools

### Images — Placid (MCP / API)
- Social graphics (announcements, guest spotlights, quote cards)
- Branded templates with dynamic text overlay
- Thumbnail generation for YouTube
- API: https://placid.app/docs/api
- MCP server available for direct integration

### Videos — Shotstack (MCP / API)
- Clip extraction from full episodes
- 9:16 vertical reformatting for Reels/Shorts
- Branded intro/outro overlay
- Text caption burn-in
- API: https://shotstack.io/docs/api
- MCP server available for direct integration

## Notes

- Graphics generated via Placid API/MCP using branded templates
- Video clips generated via Shotstack API/MCP
- Affiliate links generated by `affiliate/link-generator.js`
- Partner notification email sent automatically when folder is populated
- Folder link saved back to Episode Calendar → "Affiliate Folder Link" column
